# Workspace

Welcome to your Workspace! This folder is shown in the sidebar of your app.

## Getting Started

- Add any files or folders here and they will appear in the sidebar automatically
- Supported file types: `.md`, `.txt`, `.js`, `.ts`, `.py`, `.json`, and more
- Click the **Refresh** button in the sidebar to reload after changes

## Tips

- Create subfolders to organize your notes
- The file tree updates automatically when you add or remove files
